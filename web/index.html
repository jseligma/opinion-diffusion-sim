Opinion Diffusion Lab

This site helps you explore how social-network structure and behavioral update rules shape collective outcomes. You describe a network with NGL, describe dynamics with ORL, then run a live simulation.

How It Works

1. Specify the Network (NGL)

NGL defines who is in the population and how they are connected. You can generate networks from ER, WS, BA, or SBM families, and set initial opinion/threshold distributions.

2. Specify the Dynamics (ORL)

ORL defines how each person updates based on local influence. You can write threshold, probabilistic, and signed-influence rules with clear, auditable expressions.

3. Evolve and Compare

Build a network once, then evolve it over time. Or hold the dynamics fixed and swap network generators to compare structural effects.
